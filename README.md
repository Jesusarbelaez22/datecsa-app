# DATECSA – Sistema de Gestión de Servicios v1.0

## Instalación

1. Copiar la carpeta `datecsa` a `C:\Datecsa\`
2. Copiar el logo de la empresa a `C:\Datecsa\datecsa\assets\logo.png`
3. Abrir `C:\Datecsa\datecsa\index.html` en el navegador (Chrome recomendado)

## Acceso
- **Usuario:** Admin  
- **Contraseña:** DATECSA2026

## Módulos
- **Inicio:** Dashboard con estadísticas y alertas
- **Casos y Solicitudes:** Gestión de tickets con filtros y exportación CSV
- **Toner Disponibles:** Inventario con alertas de stock bajo
- **Registro de Llegada:** Entradas de toner al inventario
- **Toner Instalados:** Control de toners instalados en clientes
- **Órdenes de Servicio:** OS completas con diagnóstico y solución
- **Notificaciones KFS:** Centro de alertas y avisos

## Exportación
Todos los módulos tienen exportación a CSV por rango de fechas.

## Tecnología
HTML5 + CSS3 + JavaScript puro. Base de datos en localStorage del navegador.
No requiere instalación de servidor ni dependencias externas.
